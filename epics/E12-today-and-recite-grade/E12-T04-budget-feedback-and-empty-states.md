## Goal

The Today list speaks honestly when the day can't fit and goes quiet when there is nothing to do. Three calm states are wired onto the E12-T03 list from the controller's pre-built read model: (1) an **honest budget-feedback line** — one informational, autonomy-supportive line shown when the chosen scope overflows the daily time budget, offering *raise budget / lengthen cycle / pause new sabaq*, with FAR/manzil due items never dropped; (2) the **all-done** terminal surface — a calm closing line in `color.text.secondary` with no confetti, streak, badge, or exclamation mark, composed from the E10 empty-state family; (3) the **silent welcome-back** — after a gap with nothing to catch up on, the app resumes straight into the ordinary day with no "Welcome back, N days" greeting. Catch-up (the backlog re-spread, when there *is* something to catch up on) is E12-T05, not here. No string in this task carries guilt, fear, loss, a command, or "safe to drop."

## Implementation notes

This task is presentation-only: no engine call, no `review_log` write, no `DateTime.now()`, no new mutation. It renders sealed read-model variants the E12-T02 controller already computed and deep-links the budget affordances into E16-owned settings. The budget line is **copy correctness-critical for adab** → its wording (and the no-shame invariants) is pinned test-first by widget tests + the never-ship banned-phrase lint before the visuals are polished.

1. **Files** (in the `features` umbrella package, `lib/src/today/`, per eng-add-feature-module):
   - `widgets/budget_feedback_line.dart` — a dumb `BudgetFeedbackLine` taking a `BudgetFeedback` value (from E12-T02) and rendering one calm informational line + the three offered choices. No state of its own; no engine; no clock.
   - `widgets/today_all_done.dart` — composes the E10 all-done empty-state leaf (`ui-empty-state` family) with the localized closing string; pure stateless.
   - `widgets/today_silent_resume.dart` — the silent-resume variant: it renders **the ordinary day** (delegates to the E12-T03 populated/all-done body) with **no** greeting surface; in practice it is the *absence* of a banner, asserted by a golden that shows no "welcome back" chrome.
   - Wire all three into `today_screen.dart`'s `switch` over `TodayState` (the dumb View added in E12-T01/T03); add only the new branches, do not re-architect the switch.
2. **Budget line is a footer/header slot, never a section.** Place the `BudgetFeedbackLine` as a list header (above Far) or a pinned footer — it is **not** a fourth track and must never reorder or thin the Far→Near→New sections (07-components §1 anti-pattern). Render on a flat `surface`/Level 0–1 `surfaceContainer`, copy in `type.body`/`type.caption`, `color.text.secondary`; **no** red/`color.semantic.error` fill, **no** alarm icon — overflow is information, not an error (ui-daily-session-list Pattern 5).
3. **The three choices are autonomy-supportive affordances, not commands.** Render *raise budget*, *lengthen cycle*, *pause new sabaq* as quiet `TextButton`/`OutlinedButton`s (≥48dp `touch.min`) that **navigate** to the E16-owned settings surfaces (cycle-preset picker / time-budget / new-lines) via `go_router` — this task does not mutate config. Copy is phrased as options ("You can raise the budget…"), never "You must…". The "lengthen cycle" affordance's supporting microcopy, if it explains the tradeoff, traces to **C-008** ("a wider cycle is less daily work, not laziness") — no guilt for a wider cycle.
4. **FAR/manzil is never dropped — assert it, don't implement it.** The drop/keep decision is the engine's (`loadBalance`, §7.9); the View renders the `BudgetFeedback` flag and the still-mandatory Far section. Add an assertion/`Semantics` invariant that the budget line is shown **with** the Far section still fully present, never instead of it — a widget test pins "overflow state still renders every Far row."
5. **All-done copy: calm closing line, never celebration.** Compose the E10 all-done leaf with the localized string (ARB key, e.g. `todayAllDoneClosing` → "Today's revision is complete"), `color.text.secondary`, `type.body`. No confetti, no streak increment, no badge, no `!`, no success haptic (ui-empty-state Pattern 2/8). Acknowledge with copy + a `motion.duration.short` + `motion.curve.standard` fade only; honor reduce-motion (degrade to instant).
6. **Silent-resume: greet nothing.** When `TodayState` is the post-gap-but-nothing-to-catch-up case, render the ordinary populated/all-done body with **zero** welcome chrome. Do not compute "N days since last open" anywhere in the View; the controller owns the gap detection via the injected `CalendarDate`. The forbidden string "Welcome back! You haven't opened the app in N days" must not exist as an ARB key or a literal (ui-empty-state Pattern 1; C-042/C-043).
7. **Strings via the ARB pipeline, transcreated for fa/ckb/ar** (eng-add-localized-string): every line (budget overflow line, the three choice labels, all-done closing) is an `l10n.*` key in `app_ar.arb` (base) + transcreated `fa`/`ckb` (ckb pending native+scholar review); no hardcoded user-facing string; the softened-Arabic-imperative / warm-respectful-Persian register per voice §8. Any count (e.g. a budget in minutes, if shown) renders in the locale numeral set via `intl` with mixed runs bidi-isolated (FSI/PDI).
8. **RTL by geometry** (eng-rtl-and-bidi-layout): `EdgeInsetsDirectional`/`AlignmentDirectional` only; the budget line's leading text at start, choice buttons laid out start→end; the one template serves fa/ckb/ar and ckb's longer copy reflows within the same insets (ui-empty-state Pattern 9).
9. **Accessibility** (E08 conventions): one calm `Semantics` announcement per state — the budget line reads its informational sentence and its choices as buttons with labels; the all-done surface announces the closing line; the silent-resume announces nothing extra (it is the normal day). ≥48dp targets; visible focus ring on the choice buttons (07-components §6).
10. **Pitfalls to avoid:** computing budget overflow / "N days" in the widget (it's the controller's, from `buildToday`/`loadBalance` and the injected clock); making the budget line a fourth section or a red alarm; mutating cycle/budget config from this View (deep-link to E16, don't write); dropping or thinning a Far row to fit; any celebration on all-done (confetti/streak/badge/`!`/success haptic); a "welcome back, N days" greeting on resume; a "safe to drop"/"mastered"/"done" phrasing anywhere; a mandate word ("must/should/don't") in any choice label.

## Acceptance criteria

- [ ] `today/widgets/budget_feedback_line.dart`, `today_all_done.dart`, `today_silent_resume.dart` exist in the `features` package; each is a dumb stateless widget that takes a pre-built value and contains no engine call, no `DateTime.now()`, no `review_log` write (verifiable by grep over the files).
- [ ] When `TodayState` carries budget overflow, exactly **one** calm informational line renders (above/below the list, never a fourth section), offering *raise budget*, *lengthen cycle*, *pause new sabaq* as ≥48dp affordances that navigate to E16 settings — and the **Far/manzil section is still fully present** alongside it (no Far row dropped or thinned).
- [ ] The budget line uses `color.text.secondary` on a flat surface with `type.body`/`type.caption`; it uses **no** `color.semantic.error`/red fill and **no** alarm iconography.
- [ ] The all-done terminal surface is a single calm closing line in `color.text.secondary`, composed from the E10 empty-state leaf, with **no** confetti, streak, badge, exclamation mark, or success haptic.
- [ ] After a gap with nothing to catch up, the screen resumes into the ordinary day with **no** greeting surface; no "Welcome back"/"N days"/"you haven't opened" string exists as an ARB key or literal anywhere in the task's files.
- [ ] Every user-facing string is an `l10n.*` ARB key present in fa/ckb/ar (ckb flagged pending native+scholar review); no hardcoded user-facing text; any number renders in locale numerals, bidi-isolated.
- [ ] Layout is RTL by geometry (`EdgeInsetsDirectional`/`AlignmentDirectional`, no `left`/`right`); ckb's longer copy reflows within the same insets without truncating a load-bearing label.
- [ ] Each state exposes one calm `Semantics` announcement; the budget choices announce label + button role with a visible focus ring; targets are ≥48dp.
- [ ] The never-ship banned-phrase lint passes over all three locales for every string this task adds (no guilt/fear/loss, no mandate word, no "safe to drop"/"mastered"/"done", no `!`/emoji); the adab conscience pass is recorded.

## Tests

All deterministic, offline by construction (no surface here opens a socket), real bundled fonts, with the injected `CalendarDate` (never a real clock). Test-first on the copy/adab invariants.

- `features/test/today/budget_feedback_line_test.dart` (widget) — **written first**:
  - overflow state renders exactly one budget line offering the three choices, each ≥48dp and announcing a button role;
  - **FAR-mandatory invariant**: in the overflow state, every Far/manzil row from the seeded day is still present (no row dropped to fit);
  - the line uses `color.text.secondary`, not `color.semantic.error`, and renders no alarm icon;
  - tapping a choice navigates to the expected E16 settings route (a fake `go_router`/navigator records the push); **no** config mutation and **no** `review_log` write occurs (a recording repository/double asserts zero writes).
- `features/test/today/today_all_done_test.dart` (widget) — the all-done state renders the localized closing line in `color.text.secondary`; asserts **no** confetti/badge/streak widget, **no** `!` in the rendered text, and **no** success haptic call (a fake haptic boundary records only `haptic.confirm`-or-none); reduce-motion degrades the fade to instant.
- `features/test/today/today_silent_resume_test.dart` (widget) — the post-gap-nothing-to-catch-up state renders the ordinary day with **no** greeting widget; asserts the absence of any "welcome back"/"N days" text in the tree across fa/ckb/ar.
- `features/test/today/banned_phrase_test.dart` (unit) — runs the never-ship banned-phrase lint over every ARB value this task adds in fa/ckb/ar (guilt/fear/loss, mandate words "must/should/don't", "safe to drop"/"mastered"/"done"/"finished", `!`, emoji) and fails on any hit; pins that the forbidden "welcome back … N days" key does not exist.
- `features/test/today/budget_empty_states_golden_test.dart` (golden) — per-locale (fa/ckb/ar) goldens of: budget-overflow line + still-present Far section, the all-done surface, and the silent-resume body; on real bundled fonts; ckb-reflow golden proves no truncation.
- Offline guard: the suite runs under an `HttpOverrides` that fails any socket open (eng-write-dart-test) — proves these states make no network call. (The cold-start → buildToday → grade → catch-up integration journey that also exercises these states lives in E12-T09.)

- [ ] **No AI / no microphone**: nothing here records, transcribes, or detects anything; these are static informational surfaces (C2, R5).
