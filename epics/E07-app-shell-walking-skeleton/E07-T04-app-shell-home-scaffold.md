## Goal

The app renders its chrome: `app/lib/app.dart` defines `HifzApp` — a `ConsumerWidget` wrapping `MaterialApp.router` that consumes the `routerProvider` from E07-T03, declares `supportedLocales: [ar, fa, ckb]` (all RTL by construction) and `AppLocalizations.localizationsDelegates`, so app-wide `Directionality.rtl` is a *consequence of locale selection* and never a hardcoded flag. `app/lib/main.dart` is `runApp(ProviderScope(child: HifzApp()))` and nothing else (the live overrides are E07-T01/T02; the WAL open is wired there). The `HomeShell` scaffold — the `ShellRoute` body host from E07-T03 — lays out the persistent bottom navigation in RTL logical order (Today · Muṣḥaf · Mutashābihāt · Progress · Settings, rightmost = home), mirrors automatically under the ambient RTL `Directionality`, mirrors its directional nav icons via the auto-mirroring `Icons.*` family, and hosts the routed `child`. The four non-Today destinations — Muṣḥaf, Mutashābihāt, Progress, Settings — ship as **inert calm placeholder screens** (no interactive elements beyond the nav chrome): a localized title + one calm body line composing real E06 tokens (`color.*` / `type.*` / `space.*`) and fa/ckb/ar strings with `Semantics` labels, using logical start/end insets only. The Today slot is filled by E07-T07; this task ships its inert placeholder until then.

## Implementation notes

1. **`app/lib/main.dart`** — `void main()` is exactly `runApp(ProviderScope(overrides: [...], child: const HifzApp()))`. The override list and the WAL `openAppDatabase()` are owned by **E07-T01/T02** (this task does not author them); `main.dart` carries **no** other logic, no `await`-of-business work, no service read. Per project-structure §2, any computation here is a review-blocking bug.

2. **`app/lib/app.dart` — `HifzApp`** — a `ConsumerWidget`. `build` returns `MaterialApp.router(routerConfig: ref.watch(routerProvider), supportedLocales: const [Locale('ar'), Locale('fa'), Locale('ckb')], localizationsDelegates: AppLocalizations.localizationsDelegates)`, plus `theme`/`darkTheme` from the E06 token themes (by name). Do **not** wrap the tree in a `Directionality(TextDirection.rtl)` — direction is derived from the active locale by `GlobalWidgetsLocalizations` (eng-rtl §2; flutter §6). `locale` is left unset so the platform/Settings choice resolves; the `localeResolutionCallback` (if any) is deferred to E09. No `home:`/`routes:` — navigation is entirely `routerConfig`.

3. **`HomeShell` (the `ShellRoute` body)** — `app/lib/composition/router.dart` (declared by T03) hands `HomeShell(child: child)`; this task implements it as a `ConsumerWidget` (or `StatelessWidget`) returning a `Scaffold(body: child, bottomNavigationBar: <nav>)`. The nav items, **declared in logical order Today · Muṣḥaf · Mutashābihāt · Progress · Settings**, render rightmost-as-Today purely as the *visual result* of laying logical-order children under RTL `Directionality` — never a manual `.reversed`. Selected index derives from the current `GoRouterState` location; tapping an item calls `context.go('/<dest>')` (typed go_router navigation, never `Navigator.push`). Directional/leading nav glyphs use the auto-mirroring `Icons.*` family so they mirror with ambient direction (eng-rtl §2; the muṣḥaf is never an icon here). Insets are `EdgeInsetsDirectional` / `space.*` only.

4. **Four inert placeholder Views** — under `packages/features/lib/src/{mushaf,mutashabihat,progress,settings}/`, each a dumb `ConsumerWidget` named `<feature>_screen.dart` (eng-add-feature-module anatomy). Body is one calm column: a title in `type.*` (e.g. `type.titleMedium`) and one body line in `type.*` / `color.textSecondary`, centered with `AlignmentDirectional`, screen margins from `space.*`. **Inert = zero interactive elements beyond the nav chrome** — no buttons, no taps, no providers read beyond `l10n`. Today's placeholder lives in `packages/features/lib/src/today/today_screen.dart` as the same calm placeholder, to be replaced wholesale by E07-T07. No feature file imports `package:http`/`dio`/`drift`/a DAO (eng-add-feature-module §4); no `DateTime.now()`.

5. **Strings (TEST-FIRST for coverage, gate-blocking)** — author every nav label and placeholder title/body in `l10n/lib/l10n/app_ar.arb` *first* (ar = template/base), then transcreate `fa`/`ckb`/`ar` values; `ckb` canonical-encoded (U+06D5 ە, U+06A9 ک), `@description` flagged "needs native + scholar review", provisional. Read only via `AppLocalizations.of(context)` (`l10n.*`) — `nullable-getter: false` makes a missing/typo key a **compile error**. No string literal passed to `Text`/`tooltip:`/`semanticsLabel:` survives the `features/**` grep. Adab voice: reverent/calm/plain — no guilt/fear/loss, no "safe to drop"/"mastered", no exclamation marks or emoji (eng-add-localized-string §10).

6. **Accessibility** — every nav item carries a stable accessibility identifier (e.g. `nav.today` … `nav.settings`) and a `Semantics` label from the same ARB keys; each placeholder root carries a stable identifier (`screen.mushaf` …) for the E07-T10 spine journey to address. Tap targets ≥ 48 dp; Dynamic Type via `type.*` text styles only (no fixed font sizes). Selected nav state is conveyed by more than color (label + indicator).

7. **Pitfalls to avoid** — hardcoded `Directionality(TextDirection.rtl)` anywhere (RTL must come from the locale); `EdgeInsets.only(left:/right:)` / `Alignment.centerLeft/Right` / `Positioned(left:/right:)` in nav or placeholder (physical-side grep ban); manually `.reversed`-ing the nav list to "fake" RTL; a hardcoded English/Persian literal instead of `l10n.*`; raw hex/pt/ms instead of E06 token names; an interactive control on an "inert" placeholder; any `http`/`dio`/`drift`/`DateTime.now()` reaching the shell or a placeholder; redefining the route table or the redirect guard (those are E07-T03 — this task only implements `HomeShell`'s layout and the View bodies); business logic creeping into `app/lib/`.

- [ ] `app/lib/main.dart` is `runApp(ProviderScope(overrides: […T01/T02…], child: const HifzApp()))` with no other logic; a grep finds no service read, no `await` of business work, no schedule/persist/render call in `main.dart`.

- [ ] App launches on simulator/device to the `HomeShell`: a persistent bottom nav with five items; under any of fa/ckb/ar the bar renders Today rightmost (RTL) as the geometric result of logical-order children, and directional nav icons mirror.
- [ ] Tapping a nav item routes via `context.go('/<dest>')` (typed go_router) and updates the selected index from `GoRouterState`; no imperative `Navigator.push` anywhere in the shell.
- [ ] Muṣḥaf, Mutashābihāt, Progress, and Settings each render an inert placeholder (localized title + one calm body line, E06 tokens, `space.*` margins) with **no interactive element beyond the nav chrome**; the Today slot renders the same inert placeholder pending E07-T07.
- [ ] All five nav labels and all four placeholder title/body strings resolve through `AppLocalizations` (`l10n.*`) in ar/fa/ckb; a device-language switch shows the translated chrome; zero hardcoded UI literals survive the `features/**` grep.
- [ ] A physical-side grep over `features/**` and the shell finds no `EdgeInsets.only(left:/right:)`, `Alignment.centerLeft/Right`, `Positioned(left:/right:)`, or hardcoded `Directionality`; positioning is logical start/end only.
- [ ] Tokens by name only: no raw hex, no `.system(size:)`/fixed font size, no numeric `space`/`motion` literals in the shell or placeholders (E06 lint passes at error severity).
- [ ] Every nav item and placeholder root carries a stable accessibility identifier and a `Semantics` label in each locale; placeholder text scales with Dynamic Type without truncation at the largest text scale; tap targets ≥ 48 dp.
- [ ] No `package:http`/`dio`/`dart:io HttpClient`, no `drift`/DAO import, and no `DateTime.now()`/`Calendar.current` appears anywhere reachable from `HifzApp`, `HomeShell`, or a placeholder.
